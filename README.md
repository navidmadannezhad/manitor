# manitor
A Complete AI-Developed system to monitor internet usage!

We're witnessing dark days in Iranian internet, a great internet shutdown is imposed on us by Government, avoiding us to work and live properly.
Of course, we as engineers have found ways to bypass this shutdown by using different VPN methods and etc. But as VPN price is too high for us to pay (we're using VPN GB by GB, like drinking water drop by drop. Each GB of vpn usage costs us about 4$) I've implemented this system as fast as I could using AI to monitor who's using the most internet traffic, so we can use internet more "Economically!"
